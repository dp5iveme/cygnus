# LikeSwan

**Live OS 카테고리를 정의하는 기업 운영 플랫폼**

---

## 1. 전략 개요

LikeSwan은 **Live OS**라는 새로운 시장 카테고리를 정의하는 기업 운영 플랫폼이다.

| 구분 | 내용 |
|------|------|
| 카테고리 | **Live OS** (살아있는 기업 운영체제) |
| 브랜드 | **LikeSwan** (백조처럼) |

> CRM → Salesforce 처럼, Live OS → LikeSwan

---

## 2. Live OS: 새로운 카테고리

Live OS는 '살아있는 기업 운영체제'를 의미한다. 기존 Work OS가 '일을 관리'하는 데 그쳤다면, Live OS는 '기업을 운영'한다.

| 카테고리 | 핵심 기능 | 대표 플레이어 |
|----------|-----------|---------------|
| ERP | 자원을 관리 | SAP, Oracle |
| Work OS | 일을 관리 | monday.com |
| **Live OS** | **기업을 운영** | **LikeSwan** |

### Live OS 핵심 특성

- **Real-time Visibility** — Digital Twin으로 기업 전체를 실시간 가시화
- **Autonomous Execution** — AI Agent가 24/7 자동 실행
- **Human-Centric** — 사람은 사람다운 일에 집중

---

## 3. LikeSwan: 백조의 비유

백조는 수면 위에서 우아하게 떠 있지만, 수면 아래에서는 발을 열심히 젓고 있다.

| 수면 위 (우아함) | 수면 아래 (움직임) |
|------------------|---------------------|
| 사람 | AI / 시스템 |
| 창의적, 전략적, 관계 중심 | 반복 업무, 데이터 처리, 모니터링 |

> **"당신은 우아하게, 시스템은 열심히"**
